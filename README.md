## Useful commands

From the terminal/shell/command line tool, use the following commands to build, test and run the API.

- ### Build the project

```powershell
dotnet build
```

- ### Run the tests

```powershell
dotnet test
```

- ### Run the application

```powershell
# Run the application which will be listening on port `5099`.
dotnet run --project <<PROJECT_PATH>>
```
